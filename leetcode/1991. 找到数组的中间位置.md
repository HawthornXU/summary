# [1991. 找到数组的中间位置](https://leetcode.cn/problems/find-the-middle-index-in-array/)

## 粗暴双循环解决

```ts
/**
 * @param {number[]} nums
 * @return {number}
 */
var findMiddleIndex = function(nums) {
    if (nums.length === 1) {
        return 0
    }
    for (let i = 0; i< nums.length; i++) {
        let leftCount = 0;
        let rightCount = 0;
        for (let j in nums) {
            if (j < i) {
                leftCount += nums[j];
            }
            else if (j > i) {
                rightCount += nums[j];
            }
        }
        if (leftCount === rightCount) {
            return i;
        }
    }
    return -1
};
```
