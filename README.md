##

软件开发的目标不是类型安全、100%的测试覆盖率、流畅的代码逻辑、完善的开发工具、高效的系统、使用最好的编程语言、优雅的 API 设计、快速的反馈循环、编写很棒的代码等等......

软件开发的真正目标只有一个：发布解决客户问题的软件，为客户提供价值。
